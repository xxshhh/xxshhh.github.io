收录的计算机术语

发表于 2015-08-12 | 分类于 经验
本文字数： 1.4k | 阅读时长 ≈ 4 分钟

有感于自己糟糕的英语，现收录一些计算机方面的术语，以后看到会不定期补充进来，时不时可以翻阅一下。

English

注：排名不分先后，仅按时间顺序~~

  1. HTML —— HyperText Markup Language （超文本标记语言）
  2. CSS —— Cascading Style Sheets （层叠样式表）
  3. JavaScript —— （脚本语言）
  4. XML —— Extensible Markup Language （可扩展的标记语言）
  5. Ajax —— Asynchronous JavaScript and XML （异步 JavaScript 和 XML）
  6. SQL —— Structured Query Language （结构化查询语言）
  7. JDBC —— Java Database Connectivity （Java 数据库连接）
  8. Asp —— Active Server Pages （动态服务器页面）
  9. PHP —— Hypertext Proprocessor （超文本预处理器）
  10. JSP —— Java Server Pages （Java 服务器页面）
  11. W3C —— World Wide Web Consortium （万维网联盟）
  12. DAO —— Data Acess Object （数据访问对象）
  13. MVC —— Model View Controller （模型-视图-控制器）
  14. ORM —— Object Relational Mapping （对象关系映射）
  15. IoC —— Inversion of Control （控制反转）
  16. AOP —— Aspect Oriented Programming （面向切面编程）
  17. AWT —— Abstract Window Toolkit （抽象窗口工具包）
  18. DOM —— Document Object Model （文档对象模型）
  19. SOAP —— Simple Object Access Protocol （简单对象访问协议）
  20. WSDL —— Web Service Description Language （Web Service 描述语言）
  21. UDDI —— Universal Description Discovery and Integrity （通用描述、发现与集成服务）
  22. RMI —— Remote Method Invocation （远程方法调用）
  23. NDK —— Native Development Kit （原生开发包）
  24. DNS —— Domain Name System （域名系统）
  25. ARP —— Address Resolution Protocol （地址解析协议）
  26. RARP —— Reverse Address Resolution Protocol （反向地址转换协议）
  27. TCP —— Transmission Control Protocol （传输控制协议）
  28. UDP —— User Datagram Protocol （用户数据报协议）
  29. ···
