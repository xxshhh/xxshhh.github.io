整理的学习网站

发表于 2015-06-08 | 分类于 经验
本文字数： 819 | 阅读时长 ≈ 2 分钟

整理一些自己觉得实用的网站，主要是和技术相关。

程序员

IT网站

各种十分好用的IT网站。

  • GitHub - 全球知名的代码托管平台
  • API Store - 最近常常调用API，百度整理的十分丰富的API资源
  • 七牛云存储 - 图床工具，简单好用，我的博客的图片资源地址就来自于此
  • Stack Overflow - 国外最棒的的IT问答社区，google搜索IT技术问题答案关联到这个网站的比重比较大
  • 微盘 - 找资源，上微盘！用的最多的就是，用它来搜索 PDF 书籍下载。

Android相关

我经常光顾的一些Android学习网站、大牛博客等。

  • AndroidDevTools - “鲜为人知”的Android开发好网站！！！
  • 张鸿洋、hyman - 才发现常常在慕课网看的Android教学视频和这个csdn博主是一个人
  • 泡在网上的日子 - 做最好的移动开发社区

图片素材

有时候常常需要图片方面的素材，你如果百度“图片素材”你会发现那些个图片真是丑的不行······

  • 堆糖 - 分享收集生活中的美好，遇见世界上的另外一个你。
  • Dribbble - Show and tell for designers（设计师必玩）
  • 阿里巴巴矢量图标库 - 懒人必备的图标库，下载十分方便

视频学习

一些我用过的技术学习视频网站，评价是网站自带的标题，各有各的特点，不予点评···

  • 慕课网 - 国内最大的IT技能学习平台
  • 麦子学院 - 专业的IT在线职业教育平台
  • 极客学院 - 中国最大的IT职业在线教育平台
