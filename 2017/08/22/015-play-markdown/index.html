玩转Markdown

发表于 2017-08-22 | 分类于 经验
本文字数： 1.9k | 阅读时长 ≈ 5 分钟

Markdown 作为当前最流行的写作工具之一，无处不在使用，如 GitHub、CSDN、简书、Hexo 静态博客等，故细心整理一番，一起玩转 Markdown。

Markdown Syntax

Markdown 是一种轻量级标记语言。

官方语法主要有：

  • Headers - 标题
  • Emphasis - 强调
  • Blockquotes - 引用
  • Lists - 列表
  • Images - 图片
  • Links - 链接
  • Backslash Escapes - 反斜杠转义
  • 其他整理

GitHub Flavored Markdown

GitHub 使用的是它自己的一个 Markdown 语法版本，GFM。

GitHub 风格相关语法主要有：

  • Username @Mentions - 通知用户
  • Issue references - 问题参考
  • Emoji - 表情
  • Fenced Code Blocks - 代码块
  • Task Lists - 任务列表
  • Tables - 表格
  • 其他整理
